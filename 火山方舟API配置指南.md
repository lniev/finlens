# 火山方舟 API 配置指南

## 前置准备

### 1. 注册火山方舟账号

访问 [火山引擎控制台](https://console.volcengine.com/ark) 注册并登录账号。

### 2. 创建语音识别服务

1. 在火山引擎控制台搜索「火山方舟」或「语音识别」
2. 创建语音识别应用
3. 获取 API Key

### 3. 创建大模型服务

1. 在火山方舟平台创建大模型推理接入点（Endpoint）
2. 获取模型 ID（通常格式为 `ep-20241203184342-xxxxx`）

## 插件配置

### 1. 启用 AI 功能

在插件 Popup 中勾选「启用AI转录和总结」

### 2. 填写配置信息

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **API Key** | 火山方舟的 API Key | `ak-xxxxxxxxxxxxxx` |
| **API 地址** | API 基础地址 | `https://ark.cn-beijing.volces.com/api/v3` |
| **模型 ID** | 大模型接入点 ID（可选） | `ep-20241203184342-xxxxx` |

## API 说明

### 语音识别 API

- **端点**: `POST /audio/transcriptions`
- **格式**: OpenAI Whisper 兼容格式

### 大模型 API

- **端点**: `POST /chat/completions`
- **格式**: OpenAI Chat 兼容格式

## 使用流程

1. 录制音频/视频
2. 停止录制后，点击「音频转文字」按钮
3. 等待转录完成
4. 点击「内容总结」按钮获取 AI 总结

## 常见问题

### Q: API 调用失败怎么办？

A: 请检查：
1. API Key 是否正确
2. API 地址是否正确
3. 火山方舟账户是否有余额
4. 网络连接是否正常

### Q: 语音识别结果不准确？

A: 可以尝试：
1. 确保录制质量良好
2. 配置正确的语言参数（默认为中文）
3. 使用更高级的语音识别模型

### Q: 如何获取更准确的总结？

A: 可以：
1. 确保转录文本准确
2. 在设置中配置更适合的大模型
3. 调整提示词（需要修改代码）

## 注意事项

- API 调用会产生费用，请关注火山方舟的定价
- 音频数据需要上传到火山方舟服务器进行处理
- 建议在录制较短的片段后进行测试
